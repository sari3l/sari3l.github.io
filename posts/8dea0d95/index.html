i. 简单判断

  1. 选择 web 应用语言

Q&A

  1. 原理简述

    就是简单的上传一个 cmd shell

  2. 为什么要分两步呢?

    不知道，明明可以直接一步上传

  3. 使用--os-shell需要的条件

    1. FILE 权限
    2. 可写的结对路径
    3. PHP GPC off

  4. –os-shell 退出后会发生什么

    会调用后门脚本删除上传文件后，进行自删除

  5. 除了文件上传就没有了么

    注意，前面提到的是基于 PHP Web 应用的 os-shell 执行流程，更多内容自行审计看sqlmap/plugins/generic/takeover.py代码逻辑
